<h1 align="center">🎓 OgrenciOgretmenProject</h1>
<h3 align="center">Butik Eğitim Platformu</h3>

<p align="center">
Modern yazılım mimarileri ile geliştirilmiş kapsamlı bir <b>Okul / Eğitim Yönetim ve İletişim Platformu</b>
</p>

<hr/>

<h2>🎯 Projenin Amacı</h2>

<p>
Bu proje; <b>öğretmenler, öğrenciler, yöneticiler (admin)</b> ve potansiyel olarak <b>veliler</b> arasındaki iletişimi güçlendirmek ve eğitim süreçlerini dijitalleştirmek amacıyla geliştirilmiştir.
</p>

<ul>
  <li>⚡ Hızlı</li>
  <li>📱 Erişilebilir</li>
  <li>🔄 Etkileşimli</li>
</ul>

<hr/>

<h2>🚀 Temel Özellikler</h2>

<h3>🔐 Rol Bazlı Yönetim (RBAC)</h3>
<ul>
  <li>Admin, Öğretmen, Öğrenci rolleri</li>
  <li>Yetkilendirme ve erişim kontrolü</li>
  <li>Admin onaylı kullanıcı sistemi</li>
</ul>

<h3>❓ Soru - Cevap Modülü</h3>
<ul>
  <li>Öğrenciler soru yükleyebilir</li>
  <li>Öğretmenler cevaplayabilir</li>
</ul>

<h3>📋 Görev & Ödev Yönetimi</h3>
<ul>
  <li>Görev atama</li>
  <li>Durum takibi</li>
</ul>

<h3>💬 Gerçek Zamanlı Mesajlaşma</h3>
<ul>
  <li>SignalR ile anlık chat</li>
</ul>

<h3>📅 Randevu & Takvim Yönetimi</h3>
<ul>
  <li>Birebir / grup ders planlama</li>
  <li>FullCalendar entegrasyonu</li>
</ul>

<h3>📊 Audit Log</h3>
<ul>
  <li>Tüm işlemler kayıt altına alınır</li>
  <li>İzlenebilirlik sağlanır</li>
</ul>

<h2>💻 Kullanılan Teknolojiler</h2>

<h3>🧠 Backend</h3>
<ul>
  <li>.NET 9.0</li>
  <li>ASP.NET Core Web API</li>
  <li>Entity Framework Core 9</li>
  <li>ASP.NET Core Identity</li>
  <li>JWT Authentication</li>
  <li>SignalR</li>
  <li>Scalar (API Documentation)</li>
</ul>

<h3>🎨 Frontend</h3>
<ul>
  <li>Angular 21</li>
  <li>TypeScript</li>
  <li>RxJS</li>
  <li>FullCalendar</li>
  <li>SweetAlert2</li>
  <li>Zanex UI Template</li>
</ul>
